My Skills

HTML & CSS

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are the foundational technologies for building and styling web pages, enabling the structuring of content and visual presentation.

JavaScript

JavaScript is a dynamic programming language that adds interactivity and functionality to websites, allowing for the creation of responsive and engaging user experiences.

React.js

React.js is a popular JavaScript library for building user interfaces, allowing developers to create reusable components and manage the state of applications efficiently.

Node.js

JavaScript runtime that enables server-side scripting, allowing you to build scalable and high-performance web applications by utilizing non-blocking, event-driven architecture.

Express.js

Express.js is a minimal and flexible Node.js web application framework that provides robust features for building web and mobile applications, making it easier to create APIs and handle routing.

MongoDB

MongoDB is a NoSQL database that uses a flexible document data model, allowing for high scalability and performance, making it ideal for applications requiring large volumes of data and real-time updates.
